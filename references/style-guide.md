# MarkCraft Style Guide

## Design Goals

The visual direction should communicate precision, intelligence, and engineering maturity. Decoration should guide the reader through the document rather than compete with the content.

## Readability Rules

- Use 10.5-11.5 pt body text for A4 PDFs.
- Use 1.55-1.7 line height for Chinese-English mixed documents.
- Keep paragraphs visually separated with vertical rhythm rather than indentation.
- Use H1 for major report sections, H2 for subsections, and H3/H4 sparingly.
- Avoid more than 4 heading levels in final PDFs.
- Do not center long body text.
- Avoid full-page dark backgrounds except for cover pages.

## Brand Decoration Rules

- Team name should appear as a compact badge on the cover and as small header text on body pages.
- Slogan should appear on the cover and optionally in the footer.
- Use thin gradient rules, corner marks, or subtle grid patterns.
- Avoid large watermarks behind dense text.
- Never place decorative text behind code blocks, tables, or formulas.

## Content-Specific Rules

- Tables: use repeatable header styling, zebra striping, and no more than 8-10 columns per A4 page if possible.
- Code blocks: use a monospaced font and enough padding. Keep language labels if available.
- Images: constrain to page width and avoid page breaks immediately after captions.
- Citations and links: preserve original URLs. For final client-facing or formal reports, ensure URLs remain clickable if the renderer supports it.
- Chinese text: prefer PingFang SC, Microsoft YaHei, Noto Sans CJK SC, Source Han Sans SC, or WenQuanYi Micro Hei.
